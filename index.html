Building things and solving problems with bits and bytes has become the most fulfilling work I have done in my life. Projects

"Spaced Repetition"

Client Repo Server Repo Live Site

Learn a new language through the spaced repetition technique. This is a DuoLingo clone that utilizes basic data structures and a responsive front end designed in React. The back-end utlizes Node.js, Express, and PostgreSQL and supports the ability to add multiple languages to learn. Project was completed to provided test specifications and built off of a provided project base.

"Bibliomancer"

Client Repo Server Repo Live Site

Bibliomancer provides recommendations for books to read based on a selected genre. It is for those undecisive people that want to read, but are unsure what to read. Further, it tracks what books you have read from the recommendations given. It utilizes a responsive React front-end and Node.js, Express, and PostgreSQL on the back-end.

"Famous Literary Figures: What Do They Know? Do They Know Things?"

Repo Live Site

Test your knowledge of quotes from famous authors. This app uses jQuery and a Reactful design to deliver a short quiz. It was a project built for the Thinkful curriculum.
